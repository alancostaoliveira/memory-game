# 🎮 Jogo da Memória - Emoji Edition

Jogo da memória simples e divertido desenvolvido com HTML5, CSS3 e JavaScript Vanilla. O objetivo é encontrar todos os pares de emojis no menor tempo possível.

## ✨ Recursos

- Cartas geradas dinamicamente a cada partida.
- Embaralhamento aleatório dos emojis a cada carregamento.
- Regra de bloqueio para impedir mais de duas cartas abertas ao mesmo tempo.
- Feedback visual quando um par é encontrado.
- Interface leve, responsiva e pronta para abrir direto no navegador.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação dos elementos.
- **CSS3**: Estilização, layout em Grid/Flexbox e animações de transição.
- **JavaScript**: Lógica de embaralhamento, manipulação do DOM e regras de negócio do jogo.

## 🛠️ Como Jogar

1. Abra o arquivo `index.html` em um navegador moderno.
2. Clique em uma carta para revelar o emoji.
3. Clique em outra carta para tentar encontrar o par.
4. Se os emojis forem iguais, o par permanece aberto.
5. Se forem diferentes, as cartas voltam a fechar automaticamente.
6. Repita até encontrar todos os pares.

## 📁 Estrutura de Arquivos

- `index.html`: Página principal.
- `src/styles/reset.css`: Reset de estilos padrão.
- `src/styles/main.css`: Estilos visuais do jogo.
- `src/scripts/engine.js`: Motor do jogo e lógica principal.

## 🔧 Como Executar

Basta abrir o arquivo `index.html` em qualquer navegador moderno. Não há dependências externas nem etapa de instalação.

## 📌 Observação

Este projeto foi desenvolvido como estudo de JavaScript, com foco em manipulação do DOM, estados de jogo e interações simples em interface.
